ℹ️ О приложении

Калькулятор потери напряжения в кабельных линиях по упрощённой формуле ПУЭ для трёхфазных систем 380В с алюминиевым кабелем (коэффициент 30).

Для медных кабелей коэффициент составляет ~18, для однофазных 220В — используйте соответствующие справочные таблицы.
